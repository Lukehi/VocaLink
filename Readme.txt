analysis_vocalink_v1.py presents an initial exploratory analysis of Wikipedia data using two visualisations.

To run the code select which sections you would like to run from within the analysis_vocalink_v1.py code. The data collection and visualisations can be executed individually or together by selecting the appropriate options on lines 19-22.

Data is collected and saved to the current working directory. Images are saved to ./Report/Images.
